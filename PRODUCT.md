# Product

## Register

brand

## Users

Everyday Indonesian internet users — people who receive suspicious links via WhatsApp, SMS, social media, and email. They're not security professionals; they need a fast, trustworthy answer to "is this link safe?" without jargon or complexity. Context: mobile-first, often in a hurry, may have limited digital literacy.

## Product Purpose

Urlveil protects Indonesian internet users from phishing and malware by providing instant, free link analysis. Users paste a suspicious URL and get a clear safe/warn/danger verdict in seconds. Success means users trust the tool enough to check links habitually — making it a reflex, not an afterthought.

## Brand Personality

**Trustworthy, protective, approachable.** Like a knowledgeable friend who watches your back — serious about security but never intimidating. The tone is confident and clear, not clinical or corporate. Indonesian-first; speaks the user's language literally and culturally.

Voice: direct, reassuring, zero jargon.
Tone: warm but authoritative — "kami sudah cek, link ini aman" not "threat assessment: pass."

## Anti-references

- **Enterprise security tools** (Norton, Kaspersky, McAfee): cold, corporate, fear-based UIs with dashboard complexity. Urlveil should feel like a helpful tool, not a security product.
- **Generic SaaS landing pages** (cream backgrounds, gradient text, identical card grids, hero-metric templates): the saturated AI default. Urlveil should feel distinctly Indonesian and technically credible, not templated.

## Design Principles

1. **Trust earned through clarity.** Every element should make the user feel more confident, not more confused. Status colors, verdict language, and visual hierarchy must be instantly readable.
2. **Simplicity is a security feature.** Complexity breeds distrust. One input, one verdict, one action. The interface should feel as fast as the analysis itself.
3. **Distinctive warmth.** Dark themes can feel cold and clinical. Urlveil's cosmic indigo + amber palette creates warmth without sacrificing the seriousness security demands.
4. **Show, don't tell.** Demonstrate protection through live data, real-time scanning animations, and concrete threat counts — not marketing copy about how great the tool is.
5. **Polish as proof.** In security, attention to detail signals competence. Every pixel, every animation, every micro-interaction must be intentional and refined.

## Accessibility & Inclusion

- WCAG 2.1 AA compliance minimum
- Full keyboard navigation for all interactive elements
- Reduced motion alternatives for all animations
- Color-blind safe: status conveyed through icons/labels, not color alone
- Indonesian language primary; consider Bahasa Indonesia readability (short sentences, common vocabulary)
